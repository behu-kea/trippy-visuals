# Trippy Visuals (student + track overlay)

A tiny single-page website that runs trippy visuals on a full-screen canvas, while showing:

- student name
- track name

## Run it locally

### Option A: Python (recommended)

From this folder:

```bash
python3 -m http.server 5173
```

Then open:

- http://localhost:5173

### Option B: Node (if you prefer)

```bash
npx serve .
```

## Use

1. Enter **Student name** and **Track name**.
2. Click **Start visuals**.

Keyboard:

- Space = start/stop
- N = random next visual
- 1/2/3 = jump to a specific visual

## Notes

- Spacebar toggles start/stop.
